# DY01 游戏开发日志

## 2026-06-19 改动记录

### 核心系统优化

#### 程序化音频系统
- 实现完整的程序化音频生成系统，无需外部音频文件
- 支持多种音效：射击、跳跃、爆炸、击杀、受伤、死亡等
- 使用方波、正弦波、锯齿波和噪声生成不同音色
- 添加音效冷却机制防止音效过载
- 实现音量控制和淡入淡出效果

#### 物理引擎修复
- 修复地形方块之间的空隙问题
- 修复角色掉落地图底部无死亡判定的bug
- 修复敌人被子弹击中后在空中停留的bug
- 优化碰撞检测逻辑，仅在向下移动时设置grounded状态

#### 玩家系统改进
- 添加攀爬辅助逻辑，蹭到方块边缘时提供向上助力
- 修复换弹卡住的问题，改为边缘触发而非持续触发
- 实现主动换弹功能，无论子弹是否打完都可按R键换弹
- 改进死亡和复活动画效果
  - 死亡时产生更戏剧化的粒子爆炸
  - 显示"DEAD!"文字提示
  - 复活时增加绿色粒子特效和"RESPAWN!"提示
  - 复活后无敌时间延长至90帧

#### 敌人AI优化
- 改进敌人移动模式，更加自然流畅
- 添加跳跃能力，可以跳上平台
- 新增射手敌人类型
  - 橙色外观，远程攻击
  - 在50-300距离外射击玩家
  - 每1.5秒发射一次子弹
  - 子弹可被地形阻挡

#### 关卡设计改进
- 修复地形生成中的空隙问题
- 增加平台厚度，防止火箭筒轻易破坏
- 添加楼梯形状的平台，方便玩家跳跃
- 优化地图生成算法，提升可玩性

#### 视觉效果增强
- 添加子弹轨迹拖尾效果
- 实现受伤红屏闪烁效果
- 优化粒子系统性能
- 改进HUD界面设计
  - 添加连击显示系统
  - 优化血条和武器栏样式
  - 添加小地图显示功能

#### 摄像机系统
- 优化摄像机跟随逻辑
- 添加前瞻功能，根据玩家移动方向提前偏移
- 改进屏幕震动效果

#### 连击系统
- 实现连击计数器和3秒连击窗口

### 文件结构

```

├── Data/
│   ├── Config.cs          # 游戏配置和常量
│   └── Types.cs           # 数据类型定义
├── Engine/
│   ├── AudioManager.cs    # 程序化音频管理器
│   ├── Camera.cs          # 摄像机系统
│   ├── InputManager.cs    # 输入管理
│   ├── ParticleSystem.cs  # 粒子系统
│   ├── Physics.cs         # 物理引擎
│   └── Renderer.cs        # 渲染器
├── Entities/
│   ├── Bullet.cs          # 子弹实体
│   ├── Enemy.cs           # 敌人实体
│   ├── Explosion.cs       # 爆炸效果
│   └── Player.cs          # 玩家实体
├── Game/
│   ├── BuffSystem.cs      # Buff系统
│   ├── Level.cs           # 关卡生成
│   └── UI.cs              # 用户界面
└── GameMain.cs            # 游戏主逻辑
```

### 操作说明

**玩家1 (P1)**
- 移动：WASD
- 跳跃：空格
- 射击：J
- 近战：K
- 冲刺：Shift
- 换弹：R
- 切换武器：1-5

**玩家2 (P2)**
- 移动：方向键
- 跳跃：Num0
- 射击：Num1
- 近战：Num2
- 冲刺：Num.
- 切换武器：Num3-7

**通用**
- 暂停：ESC
- 返回菜单：M（暂停状态下）

### 游戏模式

1. **战役模式** - 单人闯关，消灭敌人进入下一层
2. **无尽模式** - 层层推进，敌人越来越强
3. **双人合作** - 和朋友一起战斗

### 技术栈

- **框架**: MonoGame (.NET 8.0)
- **语言**: C#
- **渲染**: 2D像素艺术风格
- **音频**: 程序化生成（无外部音频文件）
- **字体**: 支持中文字符的SpriteFont
